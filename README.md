# libshttp
 
 * A simple std c++ library for using HTTP/1.0 methods with TCP/IP socket.
 * This project is continuously working on. 

## Latest update issues
 * rewritten Makefile for libshttp.
 * removed some useless file
 * fixed some bugs.

## Latest version

## Supported methods 
 1. GET
 2. POST
 3. PUT
 4. DELETE
 5. OPTIONS
 6. HEAD
 7. TRACE
 8. CONNECT
 9. SENDFILE
 10. GETFILE
